JSON Binding with Model class using .NET Core and C#

In .NET core we can easly bind a json file with a model class using ConfigurationBinder Class. The following example will give an idea.

  • Create a new .NET core Console project in Visual Studio. Let the project name  "binding"

  • Add a new model class named Model.cs under project binding and Add two variable in Model class

            public string strVar { get; set; }

            public int intVar { get; set; }

  • Add a new json file named "mapping.json". This file entires will be mapped with model class

    {

      "Model": {

        "strVar": "Hello",

        "intVar": 999

      }

    }

    Right click the json file select properties and ensure its copied to output directory.

    Please note that the variable name in json file should mach with model class entries.

      "strVar": "Hello" -->  public string strVar { get; set; } 

      "intVar": 999 -->   public int intVar { get; set; }

  • Install the following package from nugets 

         Microsoft.Extensions.Configuration.Binder

         Microsoft.Extensions.Configuration.Json

  • Add the following code in program.cs

            var builder = new ConfigurationBuilder()...

                    .SetBasePath(Directory.GetCurrentDirectory())

                    .AddJsonFile("Mapping.json", optional: false);

                IConfiguration config = builder.Build();

                var modelClassVariable = config.GetSection("Model").Get<Model>();

                Console.WriteLine($"strVar = {modelClassVariable.strVar}");

                Console.WriteLine($"intVar = {modelClassVariable.intVar}");

                Console.ReadLine();

  • Add the necessary using 

        using Microsoft.Extensions.Configuration;

        using System;

        using System.IO;

  • Run the application and you can seethat strVar and intVar are mapped correctly with the json file.

          Output

        strVar = Hello

        intVar = 999

Program.cs

using System;

using System.IO;

using Microsoft.Extensions.Configuration;

namespace binding

{

    internal class Program

    {

        static void Main(string[] args)

        {

            var builder = new ConfigurationBuilder()

                .SetBasePath(Directory.GetCurrentDirectory())

                .AddJsonFile("Mapping.json", optional: false);

            IConfiguration config = builder.Build();

            var modelClassVariable = config.GetSection("Model").Get<Model>();//Selecting Model

            Console.WriteLine($"strVar = {modelClassVariable.strVar}");

             Console.WriteLine($"intVar = {modelClassVariable.intVar}");

            Console.ReadLine();

        }

    }

}

Model.cs

using System;

using System.Collections.Generic;

using System.Text;

namespace binding

{

    public class Model

    {

        public string strVar { get; set; }

        public int intVar { get; set; }

    }

}

Mapping.json

{

  "Model": {

    "strVar": "Hello",

    "intVar": 999

  }
